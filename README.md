# 🔍 6. Lost & Found

## "Bantu Temukan Barang yang Hilang"

### 📌 Deskripsi

Lost & Found adalah aplikasi untuk melaporkan barang hilang dan barang ditemukan di lingkungan kampus.

---

## ✨ Fitur Utama

### 📦 Posting Barang Hilang

* Nama barang
* Lokasi hilang
* Foto barang
* Deskripsi

### 📍 Posting Barang Ditemukan

* Upload barang yang ditemukan

### 🔍 Search Barang

* Cari berdasarkan nama atau kategori

### 📄 Detail Barang

* Informasi lengkap barang
* Status barang
* Kontak pelapor

### ❤️ Bookmark

* Simpan posting penting

---

## 🧠 Materi yang Diimplementasikan

✅ RecyclerView
✅ Retrofit API
✅ Room Database
✅ Fragment
✅ Navigation Component
✅ CRUD
✅ Firebase Storage (Opsional)

---

## 🎨 UI Concept

* Friendly User Experience
