# 🏠 1. KosKu

## "Temukan Kos Nyaman dengan Mudah"

### 📌 Deskripsi

KosKu adalah aplikasi mobile yang membantu mahasiswa mencari, melihat detail, dan menyimpan kos favorit secara praktis.

Aplikasi ini dirancang dengan tampilan modern dan sederhana agar pengguna dapat menemukan tempat tinggal dengan cepat.

---

## ✨ Fitur Utama

### 🔐 Authentication

* Login
* Register
* Logout

### 🏘️ List Kos

* Menampilkan daftar kos
* Foto kos
* Harga
* Lokasi
* Rating

### 🔍 Search & Filter

* Cari kos berdasarkan nama
* Filter berdasarkan harga
* Filter lokasi

### ❤️ Favorite Kos

* Simpan kos favorit
* Data tersimpan secara lokal

### 📄 Detail Kos

* Deskripsi kos
* Fasilitas
* Kontak pemilik
* Harga lengkap

---

## 🧠 Materi yang Diimplementasikan

✅ RecyclerView
✅ Fragment
✅ Navigation Component
✅ Retrofit API
✅ Room Database
✅ SharedPreferences
✅ Intent & Passing Data
